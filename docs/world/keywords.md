# 校园关键词

配置来源：`world/keywords.json`。

关键词会进入随机事件、钉钉消息和毕业总结等内容生成上下文。它们的作用不是作为硬规则，而是给本地库检索和 LLM 生成提供校园语境，让事件更像发生在求是园里。

## 维护方式

- 普通补充或修正直接编辑 `zjus-backend/world/keywords.json`。
- 新关键词建议包含 `keyword`、`category`、`desc` 和 `examples`。
- 修改后运行 `zjus-backend/scripts/validate_world_data.py`，并抽样查看算法/混合/AI 模式下的事件文本。
